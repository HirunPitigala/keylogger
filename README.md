# Keystroke Monitoring Application (Python)

## Description
This project demonstrates how keyboard input events can be captured
for educational and cybersecurity awareness purposes.

## Ethical Disclaimer
This application must only be executed on systems owned by the user
or with explicit permission. Unauthorized usage is illegal.

## Features
- Logs keystrokes with timestamps
- Runs only when started manually
- Stops on user command (ESC)
- Stores logs locally

## Platform
- Python 3
- Windows / Linux
